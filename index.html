End-to-end encrypted

ECDH key exchange with AES-256-GCM. Every conversation generates fresh keys. Runs entirely in your browser via Web Crypto API.

No trace

Close the tab and everything is destroyed. Keys, messages, identity — all gone. Nothing saved anywhere.

No account

No phone number. No email. No registration. No identity. You're nobody — by design.

No download

Works in any browser. Phone, laptop, tablet. Nothing to install. Send a link, start chatting.

Text & voice

Send text messages and voice notes up to 30 seconds. Everything encrypted end-to-end. Nothing stored. No photos, videos, or file sharing — by design.

Self-destruct

Set a timer when you create a conversation. When time's up, the room is destroyed on both sides — even if nobody closes the tab.

Open source

Every line of code is on GitHub. The crypto runs in your browser. Verify it yourself.

Blind relay

Messages pass through a relay that holds only encrypted blobs in memory — nothing is written to disk. No accounts, no logs, nothing to hand over — even with a court order. Hosted in Helsinki, EU (GDPR).
